Basic React App with Client and Server Side Rendering

## Build & Run Instructions

1. ### Clone the repo
```
    git clone https://github.com/rishabh09/react-ssr
    cd react-ssr
```

2. ### Install dependencies and devDependencies

```
    npm install
```

2. ### Run develpoment server

    ```
        npm run dev
    ```
